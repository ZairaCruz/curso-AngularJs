senaiInputText
directive in module global

Description

Diretiva que permite criar um campo de texto no padão bootstrap.

Usage

as element:
<senai-input-text
       ng-model="{ object }"
       preffix="{ String }">
</senai-input-text>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModel object

Modelo que será alterado pelo input.

preffix String

Prefixo que ficara a esquerda do input.
